# PERSONALIZED WEB PAGE

This is a personalized web application where the user enters their name and age, and the application displays content based oragew , allowing the user on input.It stores data in local storage .

## FEATURES
* User form
* Local storage
* Personalized greeting
* Age in months
* Age based access
* Motivational Quotes(loops)
* Reset functionality

## HOW IT WORKS
1. User enters their name and age
2. Data is stored in localstorage
3. Javascript processes the data
    - Displays greeting
    - Calculates age in months
    - Checks access eligibility
    - Displays motivational quotes

## TECHNOLOGIES USED
 * HTML5
 * CSS 
 * JAVASCRIPT
